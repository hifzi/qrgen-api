üñºÔ∏è Generate QR Code (PNG Image)

Endpoint: /api/qr

Method: GET

Query Parameters:

  • data (required): The data to encode in the QR code
  • size: Size in format WIDTHxHEIGHT (default: 300x300)
  • margin: Margin around QR code (0-10, default: 1)
  • errorLevel: Error correction level (L, M, Q, H, default: M)
  • darkColor: Dark color in hex (default: #000000)
  • lightColor: Light color in hex (default: #FFFFFF)

Example Request:

GET /api/qr?data=https://example.com&size=300x300&margin=2&errorLevel=H
